# Shop
-------------------

运用Struts2、hibernate和spring，以tomcat作为服务器，mysql数据库搭建的一个简单的商城应用！

前台功能包括：

    1、用户登录与注册
    
    2、浏览商品

    3、加入购物车
    
    4、生成订单并付款
    
    。。。

后台功能包括：

    1、以及分类管理

    2、二级分类管理

    3、商品管理

    4、订单管理

    。。。

这些是在学习javaWeb的时候整合的一个小项目，注释比较详细，适合新手使用
